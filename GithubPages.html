TIPS: Tạo website tĩnh với Github [free]

WorkDay
IDEAS
EVENTS
TIPS

Các bạn đang xem trang tranphatdat.net của Đạt, trang mình đang host trên Github :)

Bài này Đạt sẽ hướng dẫn cách tạo một static website là một website mà chỉ bao gồm HTML, CSS và Javascript. Bạn có thể xây dựng 1 static web hoàn toàn miễn phí phục vụ cho việc học tập hoặc thậm chí cho mục đích kinh doanh với https://github.com (chưa bao gồm tiền phí tên miền nhé :D). Sau đây là các bước thực hiện:

1. Tạo tài khoản và đăng nhập vào Github 
Bước này rất đơn giản, tương tự với hầu hết các website khác nên Đạt sẽ không nói chi tiết. 

2. Tạo mới 1 repository
Ở góc trên bên phải của bất kỳ trang nào, hãy sử dụng menu thả xuống và chọn Kho lưu trữ mới .

Các bạn chú ý, tên repository phải là dạng username.github.io (Trong đó username là tên đăng nhập của bạn, ví dụ tài khoản là laihung53 thì Đạt sẽ đặt tên repository là laihung53.github.io)

Tạo trang web của bạn

Trước khi bạn có thể tạo trang web của mình, bạn phải có một kho lưu trữ cho trang web của mình trên GitHub. Nếu bạn không tạo trang web của mình trong kho lưu trữ hiện có, hãy xem " Tạo kho lưu trữ cho trang web của bạn ."

1. Trên GitHub, điều hướng đến kho lưu trữ của trang web của bạn. 

2. Nếu bạn đang tạo một trang web dự án, hãy quyết định nguồn xuất bản nào bạn muốn sử dụng. Nếu bạn đang tạo một trang web người dùng hoặc tổ chức, bạn phải lưu trữ mã nguồn của trang web của bạn trên master nhánh. Để biết thêm thông tin, hãy xem " Giới thiệu về trang GitHub ." 

3. Nếu nguồn xuất bản bạn chọn đã tồn tại, hãy điều hướng đến nguồn xuất bản . Nếu nguồn xuất bản bạn chọn không tồn tại, hãy tạo nguồn xuất bản. 

4. Trong thư mục gốc của nguồn xuất bản, hãy tạo một tệp mới có tên index.md chứa nội dung bạn muốn hiển thị trên trang chính của trang web của bạn. 

5. Nếu bạn đang sử dụng nguồn xuất bản không mặc định cho trang web dự án, hãy định cấu hình nguồn xuất bản của bạn. Để biết thêm thông tin, hãy xem "Định cấu hình nguồn xuất bản cho trang GitHub Pages của bạn ." 

6. Dưới tên kho lưu trữ của bạn, nhấp vào Cài đặt . 

7. Để xem trang web đã xuất bản của bạn, trong "Trang GitHub", nhấp vào URL của trang web của bạn.

Chi tiết hướng dẫn tại Github

Như vậy các bạn đã có thể sở hữu một trang website rồi nhé. Bài tiếp theo Đạt sẽ chia sẻ cách mua và kết nối với tên miền, ví dụ: trang Đạt đang dưới tên miền ".net" và hiển thị đường dẫn truy cập là: https://tranphatdat.net
